# Fine-grained performance model
A fine-grained performance model of DRIM-ANN to avoid expensive index generation and testing for fast design space exploration.
## How to test
```
bash model.sh
```
## How to check the results
Check the generated excel files in the `result` folder. Please refer to the Appendix of Artifact Description/Artifact Evaluation of the SC2025 article for more details.
